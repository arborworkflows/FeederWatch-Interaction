Information

The application allows the user to add interspecies relationships to the contents of an already uploaded FeederWatch observation. The process is to enter an observation number, review the species observed, then add interactions one at a time. This application prototype has been developed by KnowledgeVis, LLC of the Arbor program in support of Cornell University.
